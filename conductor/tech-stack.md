# Tech Stack

## Programming Language
- **TypeScript:** Ensuring type safety and better maintainability across the codebase.

## Frameworks & Tools
- **React 18:** Modern UI development with hooks and concurrent features.
- **Vite:** Next-generation frontend tooling for fast development and optimized builds.
- **React Router 7:** Client-side routing for seamless navigation.
- **Vitest:** Next-generation testing framework for unit and component testing.

## Styling & Design
- **Tailwind CSS v4:** A utility-first CSS framework for rapid UI development.
- **Radix UI:** Headless UI primitives for building accessible and high-quality components.
- **Framer Motion (`motion/react`):** Declarative animations for page and component transitions.

## Content Management
- **Sanity CMS:** A headless CMS for dynamic management of committee, leadership, and event data.
- **GROQ:** Query language for fetching content from Sanity.

## Deployment & Infrastructure
- **Cloudflare Pages:** Primary hosting platform for the production frontend, providing high availability and edge caching.
